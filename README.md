#### Running Locally

To run quickr locally, follow the steps below.

1. Clone the repo

```sh
    git clone https://github.com/saifullah72437/instagram-social-media-app.git
```

2. Install All Avaialable Dependencies.

```sh
    npm install

    #or

    npm i

```

3. Start Local server

```sh
    npm start
```

You are ready to go.
